OEAPI Maintenance

This directory contains the maintenance documentation for the Open Education API (OEAPI).

It provides a central place for all information related to the ongoing upkeep of the specification and its supporting infrastructure.

Scope

The documentation in this directory focuses on:

Procedures

  • versioning and release procedures
  • change management
  • governance processes

For an overview of all procedures, see procedures.mdx.

Workflows

  • contribution workflows
  • branching strategies
  • review and approval processes

For detailed workflow descriptions, see workflow.mdx.

GitHub Environment

  • repository structure
  • branch management
  • permissions and roles
  • automation (e.g. GitHub Actions)

Configuration (future)

  • repository and organisation settings
  • tooling configuration
  • integration setup

Purpose

The purpose of this directory is to provide a clear and consistent reference for contributors and maintainers responsible for maintaining OEAPI.
